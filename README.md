# 🛡️ Auto-RPG Combat & Economy Prototype
This repository contains a Python-based simulation of an auto-battler game loop. It was developed to demonstrate **QA Methodologies**, **Edge-Case Testing**, and **Technical Documentation** for mobile game development.

## 🕹️ Project Overview
The simulation manages turn-based combat logic, random number generation (RNG) for damage, and a reward-based economy system.

## 🧪 QA & Testing Focus
As a QA-focused project, this repository includes:
* **Boundary Testing:** Health and damage thresholds.
* **Logic Validation:** Ensuring the "Marketplace" (Gold) doesn't allow negative balances.
* **Regression Testing:** Verifying that combat rounds resolve correctly every time.

## 📂 Repository Structure
* `main.py`: The core game engine and combat logic.
* `/QA_Docs`: Professional bug reports and test case documentation.
