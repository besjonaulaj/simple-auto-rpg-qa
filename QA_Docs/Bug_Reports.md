# Sample Bug Report: [BUG-001]
**Title:** Negative Gold Balance after Purchase
**Severity:** High
**Steps to Reproduce:** 1. Start game with 0 Gold.
2. Attempt to buy 'Health Potion' (Cost: 10).
3. Check Gold balance.
**Actual Result:** Gold balance shows -10.
**Expected Result:** Purchase should be blocked; error message "Not enough Gold" should appear.
